IEC 61131-3 Traffic Control System

OpenPLC Ladder Logic (LD) Safety Interlocks

Designed and simulated a comprehensive 4-way traffic control system following industrial programming standards. The focus was on moving away from rigid "cascading timers" to a robust Time-Based logic architecture.

  • Fail-Safe Logic: Implemented a 3-wire seal-in circuit to ensure immediate system shutdown on power loss.
  • Conflict Prevention: utilized Comparison Blocks (LE/GT) to mechanically prevent opposing Green lights from activating simultaneously.
  • Clean Architecture: Code structured with parallel branches for readability and troubleshooting.
